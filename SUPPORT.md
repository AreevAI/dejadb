# Support

Thanks for using **DejaDB** — an embedded memory engine for AI agents, and the
reference implementation of the Open Memory Spec (OMS).

DejaDB is a **pre-1.0 project** under active development. APIs, on-disk formats,
and behavior may change between releases. If something is unclear or broken,
please reach out — feedback at this stage is especially valuable.

## Where to get help

- **Questions, ideas, and general discussion** →
  [GitHub Discussions](https://github.com/AreevAI/dejadb/discussions). This is
  the best place for "how do I...?", design questions, and showing what you
  built.
- **Bugs and feature requests** →
  [GitHub Issues](https://github.com/AreevAI/dejadb/issues). Please use the
  issue templates so we have enough detail to help.

## Before you ask

Chances are the answer is already written down:

- The [README](README.md) — overview, install, and quick start.
- The [docs/](docs/) directory — deeper guides and reference material.
- The [FAQ](FAQ.md) — answers to common questions.

## Security issues

**Do not** report security vulnerabilities through public GitHub Issues,
Discussions, or pull requests. Please follow the process in
[SECURITY.md](SECURITY.md) instead, so the issue can be handled privately.
